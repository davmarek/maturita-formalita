9. Redakční systémy

Redakční systémy Wordpress a Joomla

Wordpress

WordPress je svobodný open source redakční publikační systém napsaný v PHP a MySQL. Má širokou uživatelskou a vývojářskou komunitu. Počet stažení předchozí verze 4.9 dosáhl od vydání téměř 174 milionů.

Uživatelsky přívětivý editor (ze šablony zvládne vytvořit web i laik) Dle oficiálních statistik je používán jako CMS na 33 % webových stránek na světě[2] a překonává tak open source CMS jako Joomla či Drupal, které se drží pod třemi procenty.

Hlavní vlastnosti:

  • Systém je dostupný zdarma
  • Integrovaná galerie médií
  • Podpora různých pluginů (téměř 50 000)
  • Widgety
  • Uživatelské účty rozdělené podle rolí (admin, šéfredaktor, redaktor atd.)

Joomla!

Joomla! je bezplatný open source CMS pro účely publikování informací na internetu. Je napsána v jazyce PHP a využívá databázi MySQL. Joomla! podporuje caching, indexaci stránek, RSS, tisknutelné verze stránek, zobrazování novinek, blogy, diskusní fóra, hlasování, kalendář, vyhledávání v rámci webserveru, lokalizace a vícejazyčné verze.

Výhody a nevýhody

Výhody

  • Samotný systém je zdarma, ovšem pokud si nechám webové stránky udělat, tak stejně musím zaplatit (SEO, grafika, tvorba). Stále vyjde ale web z redakčního systému od profesionála levněji, než web čistě udělaný v HTML, CSS a PHP
  • Možnost web udělat ze šablony - pokud si web dělám sám tak můžu zvolit jednoduchou cestu a to tvorbu webu ze šablony (U levnějších “profesionálů” dostanu stejně web z nepřizpůsobené šablony)
  • Přívětivý editor a admin sekce - S webem tak dokáže pracovat i úplný laik, jelikož je systém opravdu velmi jednoduchý. Zároveň tím usnadňuje práci profesionálním tvůrcům (web se dělá jednodušeji a nemusí řešit tvorbu editoru)
  • Pluginy - cokoliv na webu potřebuju (kalendář, e-shop, SEO atd.) existují pluginy, tudíž je tvorba opět jednodušší
  • Netřeba znát HTML, CSS a PHP - ovšem pokud si chci web přizpůsobit podle vlastních představ, tak je znát musím, ale pro obyčejné lidi co s chtějí vytvořit web zdarma to není potřeba
  • Široká komunita
  • Neustálé aktualizace
  • Lze upravovat odkudkoliv - stačí se jen přihlásit na svém prohlížeči a mám okamžitý přístup ke všem funkcím

Nevýhody

  • Nelze upravovat v takové míře HTML, CSS a PHP - Ovšem pokud chci upravovat web v takové míře tak se redakční systém stejně nevyplatí a můžu rovnou začít “od nuly”
  • Riziko napadení webu - jelikož je redakční systém jako třeba Wordpress tak rozšířený a jeho jádro je všude stejné, tak je “jednoduchý” napadnout
  • Rychlost webové stránky - nejedná se o tak výrazné zpomalení ale oproti HTML a CSS tam rozdíl je
  • Náročnost na server

Práce s programem a nastavení

U redakčních systémů bývá nějaký jednoduchý editor, kde můžu upravovat vzhled webu. Tento editor je jednoduchý i pro nezkušeného uživatele (ovšem zkušený upravuje i HTML a CSS).

Důležitou součástí je také práce s uživateli. Jako vlastník stránky můžu vytvářet nové uživatele a přidávat jim odpovídající role. Například redaktor u wordpressu nemůže zasahovat do nastavení stránek atd. A může upravovat jenom své příspěvky.

Média jsou také důležitá. Zde můžu přidávat, odstraňovat a upravovat obrázky, soubory atd.

V nastavení můžu taky nastavit třeba název stránky, formáty času atd.

Publikování článků a příspěvků

U redakčních systémů můžeme očekávat již v zákadu přidávání příspěvků. Uživatel (vlastník stránky nebo uživatel s oprávněním) může přidávat na webovou stránku příspěvky nebo články.

Klasický příspěvek má nějaký nadpis, text a náhledový obrázek. Dále můžeme v příspěvku očekávat třeba galerii, tabulky, soubory atd.

Příspěvky a články jdou také komentovat uživateli (pokud jsou tak příspěvky vlastníkem nastaveny)

Práce s moduly

Pluginy jako takové budou popsány níže v dokumentu. Ale pluginy se přidávají buď z “ obchodu z pluginy” integrovaného přímo v CMS nebo nahráním souboru (který jde stáhnout z internetu).

Šablony a webdesign

Když jdu tvořit design stránky mám více různých možností.

Buď muž použít šablonu, kterou si koupím nebo použiju šablony zdarma (Ty ale nejdou příliš upravovat a můžu pouze doplňovat text). Pokud mám šablonu, tak ji můžu upravit pomocí HTML a CSS.

Dále si můžu vytvořit vlastní šablonu (také pomocí HTML a CSS). Tato varianta je ovšem podstatně složitější a zvládne ji jen člověk co se v tvorbě webů pohybuje.

Podle mě nejefektivnějším způsobem je tvorba webu přes Elementor (který běží na wordpressu). V tomto editoru si můžu web vytvořit přesně podle svých představ a responzivní tak jak si přeju. Plugin je i cenově dostupný (licence na jeden web stojí cca 60 euro a pokud se věnuju tvorbě webů profesionálně tak stojí licence cca 150 euro na rok na neomezený počet webů)

Menu

Tvorba menu je jednoduchá. V systému si ho můžu vytvořit a na stránce nastavit, kde se má jaké menu zobrazovat. Pokud tak změním menu v nastavení stránky, tak se automaticky změní všude, kde se menu zobrazuje.

Rubriky a štítky

Příspěvky a články můžu dělit do různých rubrik a přidat jim štítky. Na stránce s příspěvky si tak můžu zobrazit příspěvky podle toho do jaké rubriky patří nebo podle toho jaký štítek má daný příspěvek.

Důležité pluginy

Elementor

Elementor je plugin pro wordpress, který umožňuje tvorbu designu webových stránek. Díky elementoru si můžu vytvořit web podle svých představ a to i bez znalosti HTML a CSS.

Woocommerce

Woocommerce je plugin také pro wordpress. Jedná se o plugin který umožňuje tvorbu e-shopu. Nabízí skladový systém, platbu a celkově funkce, které e-shop potřebuje.

JCE

Plugin pro Joomlu, ve kterém můžu upravovat vzhled webu. Uživatelským prostředím je podobný systému jako je třeba word.

E-shopy

E-shop můžu vytvořit třeba ve wordpressu přes plugin woocommerce. Je to asi nejpoužívanější varianta pro tvorbu e-shopů. Woocommerce je totiž v základní verzi zdarma a je uživatelsky přívětivý. Pokud potřebuju ale profesionálnější funkce, než jsou ty základní, tak musím koupit placenou verzi (Tato verze je i přes to poměrně levná).

Pro platbu online musím použít platební bránu, která už bývá placená fixní částkou + procenty z dané částky.

Ve woocommerce je také skladový systém (přehled zásob), prodej online obsahu, správa objednávek atd.

K woocommerce můžu použít také pluginy na tvorbu faktur a podobné rozšiřující pluginy.

Pokud dělám jen čistě e-shop, tak je dobrý také PrestaShop. Prestashop podporuje také platby online, správu skladu atd. Aktivně tento plugin používá cca 250 000 e-shopů. Jedná se o profesionálnější CMS pro e-shopy než woocommerce, jelikož je zaměřený čistě jen pro to. Ovšem nemá tak rozšířenou uživatelskou základnu a je složitější v něm pracovat.
